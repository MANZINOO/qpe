# Break Even Point - QPE

## Costi Fissi Mensili

| Voce | Costo |
|------|-------|
| Firebase Blaze (pay-as-you-go) | ~0 EUR (free tier sufficiente per MVP) |
| Dominio .com | ~1 EUR/mese |
| Vercel Pro (se necessario) | 0 EUR (free tier) |
| **Totale costi fissi** | **~1 EUR/mese** |

## Costi Variabili
- Firebase oltre free tier: ~0.06 EUR per 100K letture Firestore
- Bandwidth aggiuntivo: ~0.12 EUR/GB oltre 10GB

## Ricavi Previsti

### Sondaggi Sponsorizzati
- Prezzo per sondaggio sponsorizzato: 50-200 EUR (dipende dalla reach)
- Target: 2-5 sondaggi sponsorizzati/mese nel primo trimestre

### Insight Dashboard per Brand
- Abbonamento mensile: 99-299 EUR/mese
- Target: 1-3 brand nel primo semestre

## Calcolo Break Even
Con costi fissi di ~1 EUR/mese, il break even si raggiunge con il primo sondaggio sponsorizzato. Il modello e sostenibile fin dal primo cliente.
